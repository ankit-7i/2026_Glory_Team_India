# ⚡ Deployment

This project is deployed using **Vercel**

### Deploy Steps

1️⃣ Push code to GitHub
2️⃣ Open Vercel
3️⃣ Import repository
4️⃣ Click Deploy

# 💻 Run Locally

Clone repository

```bash
git clone https://github.com/ankit-7i/2026_Glory_Team_India.git
```

Go to project

```bash
cd 2026_Glory_Team_India
```

Open

```
index.html
```

in your browser.
